== Description ==


It is a plugin that consists of putting a personalized cookie notice (text + image) on the web page. it is easy to install and configure


== Installation ==
 
Search for `Cookies Privacy Policy` under Plugins on your WordPress install or download and:

1. Upload `cookies-policy-privacy` to the `/wp-content/plugins/` - not in format ZIP - directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or 

1. Download `cookies-policy-privacy` format ZIP
2. Add new plugin 
3. Upload plugin
